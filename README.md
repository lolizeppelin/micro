# 文件夹说明

- breaker 熔断器
- broker 异步推送(默认kafka)
- breaker grpc客户端
- codec 序列化工具
- errors 通用错误
- interfaces 通用接口
- log 日志
- registry 注册服务
- resolver 
- selector 选择器
- tracing 链路追踪
- transport grpc传输层
- utils 通用工具
